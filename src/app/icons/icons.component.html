Icons

Epy Font Icons

To use the icon font library import the following css on your header:

    <link rel="stylesheet" href="https://storage.googleapis.com/assets-front/fonts/epy-icons.css"/>
Classes

You can use an icon class like this: <span class="epy-icon-acount-settings"></span>

also change the color of the icon with the 'color' css property.
